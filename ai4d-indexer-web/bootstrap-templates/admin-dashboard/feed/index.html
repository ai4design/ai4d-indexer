Portal – Free Bootstrap 5 Admin Dashboard Template for Developers

Portal is a free Bootstrap 5 admin dashboard template made for developers. The design is simple and modern and the code is written in a modular way. It’s the perfect starter template for building your web app’s admin portal. All the source SCSS files are included so you can change the color scheme and fine tune the design components to suit your app’s branding. Built on the latest Bootstrap 5 alpha version this template is lightweight (no jQuery, only vanilla JS) and future-ready.

Want to see more page designs for this template? Please submit your design request via Github’s issues section. We will consider popular requests in future versions.
