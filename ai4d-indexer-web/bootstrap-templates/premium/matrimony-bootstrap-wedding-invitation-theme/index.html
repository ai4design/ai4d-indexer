Bootstrap 5 Template for Weddings

Matrimony is a Bootstrap 5 wedding invitation template. It comes with 8 hand-picked color schemes and you can change the look and feel of the template by changing the font or color to suit the wedding. The source SCSS files are included so making styling customisations is super quick and easy.
